## 💻 How It Works

1. **User enters a city** → "London"
2. **App fetches data** → Open-Meteo API (Geocoding)
3. **Coordinates obtained** → Forecast API call
4. **Data is processed** → Converted to daily summary
5. **Insights generated** → Best day, alerts analyzed
6. **UI updates** → Charts, tables, recommendations displayed

## ❓ Troubleshooting

### Problem: "npm: command not found"
**Solution:** Install Node.js from nodejs.org

### Problem: Port 3000 already in use
**Solution:** 
```bash
npm start -- --port 3001
```

### Problem: Module not found error
**Solution:**
```bash
npm install
```

### Problem: App won't start
**Solution:**
1. Close the terminal (Ctrl + C)
2. Run: `npm install`
3. Run: `npm start` again

### Problem: Chart doesn't show
**Solution:**
- Refresh the browser (F5)
- Clear browser cache (Ctrl + Shift + Delete)
- Wait a few seconds for it to render

### Problem: API returns error
**Solution:**
- Check your internet connection
- Verify city spelling
- Try a major city first (e.g., "London")

**For more help:** See SETUP.md → Troubleshooting section
